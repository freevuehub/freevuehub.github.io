```javascript
// isUndefined
// 지정된 값이 정의되지 않은 경우 true를 반환하고 그렇지 않으면 false를 반환합니다.
//
// 완전 항등 연산자를 사용하여 값과 val이 undefined와 같은지 확인하십시오.

const isUndefined = val => val === undefined;

// 예시
isUndefined(undefined); // true
```
